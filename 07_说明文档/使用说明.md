# 车牌识别项目说明

本项目目标是：输入车辆图片，定位车牌，切分字符，再用训练集训练出来的模型识别车牌号码。

这里不调用现成 OCR 识别库，而是使用 OpenCV 的 SVM 分类器训练自己的字符识别模型。

## 文件

- `license_plate_app.py`：主程序，负责车牌定位、字符切分、加载模型识别
- `train_character_model.py`：训练程序，负责用训练集训练字符分类模型
- `test_images`：测试车辆图片
- `models`：训练后自动生成，保存模型文件
- `output`：识别后自动生成，保存中间结果和识别结果

## 训练集格式

训练集按“一个类别一个文件夹”的方式放置，例如：

```text
train_chars/
  京/
    001.png
    002.png
  沪/
    001.png
  A/
    001.png
    002.png
  B/
    001.png
  0/
    001.png
  1/
    001.png
```

每张图片最好是单个字符图片，黑底白字或白底黑字都可以，程序会统一预处理成 `20x40`。

## 安装依赖

```powershell
py -m pip install opencv-python numpy
```

## 训练模型

在项目目录运行：

```powershell
py train_character_model.py -d train_chars
```

训练完成后会生成：

```text
models/plate_char_svm.xml
models/plate_char_labels.json
```

## 识别车辆图片

```powershell
py license_plate_app.py -i test_images -o output
```

如果模型存在，程序会输出 `text=识别结果`，并在每张图片的输出文件夹里保存：

- `plate.png`：定位到的车牌
- `binary.png`：二值化后的车牌字符区域
- `segmentation.png`：字符切分结果
- `char_01.png` 等：切出的字符
- `result.txt`：识别出来的车牌文本

## 下一步重点

如果训练集还没有准备好，可以先用当前程序切出来的 `char_*.png` 人工分类到 `train_chars` 里，形成第一版训练集。训练集越多、越干净，识别效果越稳定。
